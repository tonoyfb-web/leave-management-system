# Leave Management System

A comprehensive web-based Leave Management System with role-based access control for employees, managers, and administrators.

## Features

### For Employees
- View leave balance dashboard with visual progress bars
- Apply for leave with automatic working days calculation
- Check available balance before applying
- View leave history and status
- Cancel pending requests
- Export personal leave data to Excel

### For Managers
- View all pending requests from their department
- Approve requests with one click
- Reject requests with reason
- View team leave statistics
- Export team leave reports

### For Administrators
- **Employee Management**: Add, edit, delete employees
- **Department Management**: Manage departments
- **Designation Management**: Manage job designations
- **Leave Type Management**: Configure leave types and default days
- **Leave Balance Management**: Edit employee leave balances
- **Complete Export**: Export all data to Excel

### Notification System
- Real-time notifications for leave applications
- Toast popups for important events
- Notification panel with unread count
- Role-based notification filtering
